Návod k použití

Pro využití možnosti nahrávání pomocí tohoto formuláře potřebujete tzv. GitHub "access-token", který je možno si vygenerovat po přihlášení do GitHUB zde.

Použijte volbu "Generate new token", zadejte popisný název pro token, a ze zaškrtávací nabídky vyberte POUZE "public_repo" - Access public repositories , NIC jiného. Vygenerovaný dlouhý řetězec si uložte na bezpečné místo (např. do správce hesel). Budete ho zadávat při každém vyplňování formuláře. Vzor vyplnění najdete zde.

Tento token je třeba zadat do kolonky níže. Má stejnou váhu, jako přihlášení heslem, je třeba ho náležitě chránit a uchovávat v tajnosti. V případě potřeby lze token vymazat na odkazu uvedeném výše. Tamtéž lze i vyrobit nový.
